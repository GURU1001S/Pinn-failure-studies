--- exp19_burgers_failure_map.py ---
exp19_burgers_failure_map.py — Experiment 19: Failure Mode Phase Diagram for Burgers PINNs

Systematically sweeps viscosity ν from 0.1 to 0.0001 (20 log-spaced steps) on the
viscous Burgers equation and classifies the failure mode at each ν.

Failure mode taxonomy:
  (A) successful_convergence    — L2 < 0.05, loss stable
  (B) spectral_bias             — high-frequency Fourier error dominant
  (C) gradient_pathology        — |∇L_pde| / |∇L_bc| > 100 (gradient imbalance)
  (D) collocation_starvation    — residual concentrates near shock; sparse collocate

exp20_failure_fingerprinting.py — Experiment 20: Failure Fingerprinting & Clustering

Novel failure fingerprinting experiment across 4 PDEs.
For each PDE, 10 PINNs are intentionally driven to failure via parameter tuning.
Each failed run is characterised by 8 diagnostic signals then clustered (k-means, k=4).

PDEs tested:
  1. Advection     — β ∈ {30,50,80}            (high β → spectral failure)
  2. Burgers       — ν ∈ {0.001, 0.005, 0.0001} (low ν → shock / gradient failure)
  3. Allen–Cahn    — ε ∈ {0.01, 0.005}          (small ε → sharp interface failure)
